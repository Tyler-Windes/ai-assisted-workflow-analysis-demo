#### Validation Prompt

Review the structured workflow record for potential quality issues.

##### Check For

- missing required fields
- unclear ownership
- inconsistent priority
- duplicate indicators
- missing resolution notes for closed items
- vague descriptions
- system-area naming issues

##### Return Fields

- validation_status
- issues_found
- recommended_next_action
- human_review_required

##### Rules

- Do not resolve ambiguity by inventing facts.
- If evidence is incomplete, mark the record as needing review.
- Do not make final prioritization decisions.
