#### User Stories

##### Operations Manager

- As an operations manager, I want workflow records to use standardized priority values so that SLA risk can be measured consistently across teams.
- As an operations manager, I want incomplete records to be flagged so that follow-up work can be prioritized.
- As an operations manager, I want duplicate indicators preserved so that repeated intake issues can be reviewed before records are merged.

##### Technical Analyst

- As a technical analyst, I want raw intake values preserved so that normalization decisions remain traceable.
- As a technical analyst, I want system-area values standardized so that reports can group related issues consistently.
- As a technical analyst, I want closed records without resolution notes flagged so that closure quality can be reviewed.

##### Reporting User

- As a reporting user, I want clean status and priority values so that KPI summaries are consistent.
- As a reporting user, I want missing ownership identified so that handoff gaps can be measured.

##### Review Analyst

- As a review analyst, I want assisted classification outputs marked as draft so that final decisions remain accountable and reviewable.
